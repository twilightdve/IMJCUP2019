• 出場は全14チーム。
  • 予選は2つのグループリーグで実施し、予選の結果に応じてノックアウト方式の決勝トーナメントを行い順位を決める。

予選リーグ

  • 7チームずつ、2つのグループに分かれる。
  • 試合時間：8分
  • 入れ替わり時間：2分

決勝トーナメント

  • 上位トーナメント、下位トーナメントからなる。
  • 予選各リーグの上位3チームは上位トーナメント、下位4チームは下位トーナメントに振り分ける。
  • 試合時間：8分
  • 入れ替わり時間：2分
  • 1チーム同時に出場できるのは5人。（フィールドプレイヤー4人、キーパー1人）
  • メンバーの交代は無制限です。交代選手がコートの外に出てから、入れ替え選手はコートの中に入りましょう。
  • 女性は6人目の選手としてプレー可能です。ただし、得点しても1点のカウント。また、経験者は対象外。
試合開始時
  • ジャンケンで勝った方のボールで、キックオフ。
  • キックオフ直後のシュートは危ないのでやめましょう。
タッチラインを割った場合
  • 相手ボールのキックイン
ゴールラインを割った場合
  • 相手のゴールラインを割った → 相手ボールのゴールクリアランス
  • 味方のゴールラインを割った → 相手ボールのコーナーキック
※ゴールクリアランスとは、キーパーが手でボールを投げて再開する行為。
ゴールした場合
  • 失点したチームのキックオフで試合再開。
  • 反則を犯した場合は、相手ボールとなり、ファールを受けた場所からの直接フリーキックでプレーが再開されます。
  • 特に自陣のペナルティエリア内で反則を犯した場合は、相手ボールのペナルティキックとなります。

●キッキング：相手を蹴ったり蹴ろうとする行為

●トリッピング：相手をつまづかせたり、つまづかせようとする行為

●ファールチャージ：無謀に、あるいは過度の力で相手にぶつかる行為

●ファールタックル：タックルをした時、ボールに触れる前に相手選手に触れる行為

●ホールディング：相手選手を抑える

●ハンドリング：意図的に手でボールに触れる

  • キーパー以外のプレイヤーは、オンプレーの際はいかなる場合も手を使ってはいけない。
  • キーパーは、自陣のゴールエルア内のみ手を使うことができる。ただし、味方から受けたパスについては、キーパーは手を使ってはいけない。

●その他

  • 遅延行為 / 故意な時間稼ぎ
  • キックオフシュートや故意に相手に強くボールを蹴りつけるなど、危険と判断できる行為
  • 暴言などの侮辱行為（主に、相手および審判、観客に対するもの）

▼警告（イエローカード）となる行為

  • 反スポーツ的行為
  • 言葉または行動で異議を示す
  • 繰り返し規則違反をする
  • プレーの再開を遅らせる
  • 主審の許可を得ずフィールドに入ったり出たりする

▼退場（レッドカード）となる行為

  • 著しく不正なプレー
  • 乱暴な行為
  • 相手プレーヤーや、その他の人につばを吐く
  • 同じ試合で２つの警告を受ける（イエローカード2枚）
  • 攻撃的、侮辱的で下品な発言や行動をする
  • 相手チームの決定的な得点の機会を阻止す

※退場者が出た場合

  • 退場者が出た試合の残り時間は、一人少ない状態で試合を続ける。
  • 退場者は、退場した試合の交代出場は不可。さらに、次の試合も出場停止

▼ファールとならない行為

●バックパス

  • 味方キーパーへパス（バックパス）はOKです。
  • ただし、明らかな時間稼ぎを目的としたバックパスはやめましょう。

▼アドバンテージ（ファールがあったがプレーを流す行為）

  • 反則を受けたチームが、プレーを続けたとき有利になる場合にプレーを流す。
